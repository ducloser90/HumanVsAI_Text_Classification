# 🤖 Human vs Machine Text Classifier

A Streamlit web app that classifies whether a text was written by a human or generated by AI, using a Random Forest classifier trained on TF-IDF features.

## Project Structure

```
HumanVsAI_Text_Classification/
├── app/
│   └── app.py              # Streamlit web application
├── src/
│   ├── __init__.py
│   ├── config.py           # Central configuration & hyperparameters
│   ├── data.py             # Dataset loading utilities
│   ├── features.py         # TF-IDF vectorizer builder
│   ├── model.py            # Model training & evaluation
│   └── artifacts.py        # Save / load / inference helpers
├── scripts/
│   └── train.py            # Training pipeline entry point
├── docker/
│   └── Dockerfile
│   └── requirements.txt
├── data/                   # JSONL dataset files (not tracked)
│   ├── train_data.jsonl
│   ├── dev_data.jsonl
│   └── test_data.jsonl
├── saved_model/            # Trained model artifacts
│   ├── vectorizer.pkl
│   └── rf_model.pkl
├── requirements.txt
├── .gitignore
└── README.md
```

## Setup

```bash
# 1. Clone the repo
git clone https://github.com/ducloser90/HumanVsAI_Text_Classification.git
cd HumanVsAI_Text_Classification

# 2. Create and activate virtual environment
python -m venv myenv
source myenv/bin/activate  # Windows: myenv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt
```

## Train the Model

```bash
# Run from project root
python scripts/train.py
```

## Run the App

```bash
# Run from project root
streamlit run app/app.py
```

## Docker

```bash
# Build
docker build -f docker/Dockerfile -t human-vs-machine .

# Run
docker run -p 8501:8501 human-vs-machine
```
